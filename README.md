## セットアップ
- SwiftLint のインストール
```
$ brew install swiftlint
```

- gem のインストール
```
$ bundle install --path=vendor/bundle
```

- カラーパレットのインストール
```
$ make install
```

- ライブラリのインストール
```
$ pod install
```
